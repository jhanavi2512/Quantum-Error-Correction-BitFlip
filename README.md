# Quantum Error Correction – Bit Flip Code
This project demonstrates a simple simulation of Quantum Error Correction using the Bit Flip Code.
## Objective
To simulate how quantum information can be protected from bit flip errors using redundancy.
## Tools Used
- Python
- Qiskit
- Jupyter Notebook
## Project Steps
1. Encoding a logical qubit into three physical qubits
2. Introducing a bit flip error
3. Detecting the error using CNOT gates
4. Measuring the results
## Files in this Repository
bit_flip_code.ipynb – Simulation code  
explanation.md – Theory explanation
